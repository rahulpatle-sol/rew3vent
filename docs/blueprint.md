# **App Name**: Rew3vent

## Core Features:

- Wallet Connection: Allow users to connect their wallet to the platform. if user connect wallet or wallet connection is successful show the message
- Event Posting: Enable users to create and post event listings, including details such as date, time, location, description, and rewards.
- Event Browsing and Registration: Provide a searchable listing of events for users to browse and register to attend.
- Attendance Tracking: Maintain a record of events attended by each user.
- Calendar Event System: Provide a calendar-based event system that users attended or register for.
- Explore event: Explore event near the user
- Subscribe to event host: User can subscribe the event host user get new notification from the host
- Subscriber count: If user host an event user have subscribers count page this show the total subsciber
